# Pong Game

A simple Pong game implemented using Pygame.

## Requirements

- Python 3.x
- Pygame library

## Usage

1. Clone the repository:
    ```sh
    git clone https://github.com/brkahmed/Pong-Game.git
    cd Pong-Game
    ```
2. Install the required dependencies:
    ```sh
    pip install pygame
    ```
3. Run the game:
    ```sh
    python pong.py
    ```
